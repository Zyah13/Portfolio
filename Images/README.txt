Image Files Needed:
===================

Please add the following image files to this folder:

1. profile.jpg - Profile photo for the home section (recommended: 500x500px, square)
2. about.jpg - Photo for the About Me section (recommended: 600x600px, square)
3. project1.jpg - Screenshot/image for Web Application project (recommended: 800x600px)
4. project2.jpg - Screenshot/image for System Module project (recommended: 800x600px)
5. project3.jpg - Screenshot/image for UI Design project (recommended: 800x600px)

All images will automatically have purple glow effects applied.
You can use any image format (jpg, png, jpeg, webp).
